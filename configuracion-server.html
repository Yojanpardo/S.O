Antes de configurar el server

Ubuntu server es un sistema operativo que no tiene una interfaz grafica de usuario, por lo que tenemos que hacer todo desde comandos. Primero que todo debes tener instalado ubuntu server, si no lo tienes instalado aqui puedes ver como descargar e instalar ubuntu server

1. Actualizamos los repositorios y el sistema operativo

Para esto hacemos escribimos "sudo apt-get update" le damos enter, este proceso no tardara mucho. cuando termine escribimos "sudo apt-get upgrade" este proceso tardara un poco mas pero es necesario para que nuestro servidor esté actualizado hasta la fecha.

2. Instalamos los programas necesarios

Para poder hacer la conexión entre nuestro servidor y nuestra maquina cliente debemos instalar unos programas que son apache2 y mysql-server para ello, en el server escribimos "sudo apt-get install apache2" esperamos a que termine de instalar, luego instalamos mysql-server, para lo cual escribiremos "sudo apt-get install mysql-server" esperamos un momento a que se instale y continuamos.

3. Configurando la red interna

Esta parte es bastante delicada puesto que debemos configurar de forma manual nuestro servidor, lo que hacemos es editar el archivo interfaces que se encuentra en la ruta /etc/network/interfaces para poder editarlo tenemos que hacerlo como superusuario. Tecleamos el siguiente comando
"sudo nano /etc/network/interfaces"
aquí cambiamos los valores que tiene por defecto y le asignamos los que queramos, para este caso practico yo lo configuré asi como aparece en la imagen

4. Red interna de la maquina virtual

En este punto ya hemos terminado lo que es la configuración del servidor, ahora para poder ver lo que tenemos en nuestro servidor lo que tenemos que hacer es configurar nuestro cliente para que pueda ver los archivos que tenemos en el servidor

5. Configurando el cliente

Si queremos ver lo que tenemos en el servidor tenemos que configurar unos parametros en el cliente. Para empezar configuramos nuestras conexiones de red aqui como pueden ver en la imagen.

Como estamos en una maquina virtual detecta la red a internet como una conexión cableada por lo que aparecerá lo que ven en pantalla, la seleccionan y le dan en editar

7. COnfigurando la ip

En metodo seleccionamos manual, si seguiste esta guía al pie de la letra pones la dirección asi como la tengo y si no pues le pones la dirección que le diste al server pero le cambias el ultimo numero, mascara de red le pones 255.255.255.0, no te asustes si se cambia por un 24, es normal. Le damos en guardar.

8. Finalizando

Si todo salio correctamente al poner http://"la ip de tu servidor".
