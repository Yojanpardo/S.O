Antes de la instalación

Ubuntu Seerver es un sistema operativo tipo servidor sin interfaz grafica, más que todo es usado para servidores web. Un servidor es una maquina que nos proporcionaalgún servicio, puede ser de bases de datos, servidor de archivos, etc.

Requisitos mínimos

  • * 256MB de RAM (recomendado 1GB)
  • * Disco duro 2GB de espacio disponible (recomendado unos 40GB para poder manter unos 25 usuarios.)
  • * Procesador AMD o Intel 32 o 64 bits (recomendado dualcore a 2.4GHz)

Aquí puedes descargar Ubuntu server
1. Idioma

Despues de insertar nuestro disco o usb con ubuntu server y encender nuerstro pc nos aparecera una interfaz en la cual no veremos el mouse, haremos todo desde el teclado y lo primero es seleccionar el idioma en el que vamos a instalar nuestro Ubuntu server, para hacerlo nos desplazamos con las flechasde arriba y abajo o la tecla tab, luego precionamos enter

2. Inicia la instalación

Despues de seleccionar el idioma nos aparece la interfaz que vimos detras de esta tarjeta, le damos en instalar ubuntu server (la primera opción), presionamos enter para continuar.

3. Zona horaria

A continuación seleccionamos nuestro país para que ubuntu server reconozca nuestra fecha y zona horaria.

4. Configurando el teclado

Una vez terminado el proceso de formato, el programa de instalación hace una revisión de los discos para asegurarse de que se formatearon de forma correta y así prevenir futuros errores del sistema

5. Nombre de la máquina

Ingresamos el nombre que le vamos a dar a nuestra maquina.

6. Nombre de usuario

Ingresamos nuestro nombre de usuario. Si, es diferente del nombre de la maquina.

7. COnfigurando contraseña

A nuestro usuario recien creado le asignamos una contraseña, damos en continuar, la verificamos y nuevamente en continuar.

8. Cifrando información

El cifrar la carpeta personal da un plus a la seguridad de nuestro server pero yo recomiendo no cifrarla ya que al tenerlo cifrado a la hora de hacer busquedas es algo más lento el acceso a la información pero si lo deseas puedes cifrar tu carpeta personal.

9. Configurando la hora

Si nuestra zona fue configurada correctamente aqui le damos en si pero si no configuramos la hora manualmente, generalmente el instalador hace todo automaticamente.

10. Particionando el disco

Si estamos instalando el server en una maquina que está destinada exculsivamente para el server seleccionamos la segunda opción esto para que haga las particiones lógicas en el disco duro.

11. Seleccionando el disco

Seleccionamos el disco que queremos particionar y oprimimos enter para continuar.

12. aceptando los cambios

Selecionamos si para que se empiecen a particionar los discos y la instalación del sistema comience.

13. Tamaño del disco

Seleccionamos todo el tamaño del disco y el intalador hará automaticamente todo el proceso departicionamiento.

14. Particiones LVM

LVM o Logical Volume Manager. Aquí el asistente de particionamiento logico nos indica que es lo que va a suceder a continuación, selecionamos si y le damos enter para continuar.

15. Instalando el sistema

Si todo salió correctamente vamos a ver una barra de carga que dice que se está instalando el sistema operativo. Solo debemos esperar a que termine y listo.
