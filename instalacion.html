Antes de la instalación

Windows XP es un sistema operativo bastante liviano y con pocos requerimientos de máquina, esto lo hace ideal para poder hacer su instalación en un equipo que pueda estar algo viejo.

Requisitos mínimos:


* Procesador Pentium 233 MHz o superior (de preferencia de 300 MHz).
* Mínimo 64 MB de RAM (de preferencia 128 MB).
* Mínimo 1,5 GB de espacio disponible en disco duro.

Aquí puedes descargar Windows XP SP3
1. Asignación del espacioadd
1. Asignación del espacioclose

En este primer paso lo que hacemos es decirle al sistema operativo donde instalarse.
Podemos redimensionar el espacio del disco al tamaño que querramos e instalar el sistema operativo ahí o simplemente asignarle todo el espacio de golpe, que en este caso fue lo que hice.
Presionamos Enter para continuar

2. Selección del formatoadd
2. Selección del formatoclose

A continuación seleccionamos el formato que le vamos a dar al disco donde se va a instalar Windows, Windows solo puede leer correctamente discos que estén en formato NTFS, FAT y FAT-32. Estos son los que nos muestra, seleccionamos NTFS y presionamos Enter para continuar con la instalación

3. Dando formatoadd
3. Dando formatoclose

En este punto solo debemos esperar a que Windows de el formato selecionado a la partición para poder hacer la copia de los archivos necesarios para la correcta ejecución del sistema operativo.

4. Examinando los discosadd
4. Examinando los discosclose

Una vez terminado el proceso de formato, el programa de instalación hace una revisión de los discos para asegurarse de que se formatearon de forma correta y así prevenir futuros errores del sistema

5. Copiando los archivosadd
5. Copiando los archivosclose

Una vez terminada la revisión de los discos, se empieza con el proceso de copia de los archivos necesarios y fundamentales para el correcto funcionamiento del sistema operativo, entre estos archivos que se estan copiando encontramos la aplicación de escritorio, el archivo boot.ini, el explorador, drivers, entre muchos otros fundamentales.

6. Instalando Windowsclose
